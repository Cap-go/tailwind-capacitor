# Tailwind Capacitor Example App

This example demonstrates all the major components and features of tailwind-capacitor.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:3000`

## Features Demonstrated

- **Buttons**: Solid, outline, and clear variants
- **Lists**: Interactive list items with icons and chevrons
- **Forms**: Input fields, toggles, and checkboxes
- **Cards**: Content containers with actions
- **Badges & Chips**: Small status indicators
- **Progress & Range**: Interactive controls
- **Modals**: Toasts and dialogs
- **Navigation**: Navbar and floating action button

## File Structure

- `src/App.vue` - Main application showcasing all components
- `src/main.js` - Vue app entry point
- `src/style.css` - Tailwind CSS imports with @plugin directive
- `vite.config.js` - Vite build configuration
